# 48 - Enhancement (Spatial-Audio): Preset-Menü nach Sound-Klasse filtern

## 1. Why

### 1.1 Objective
Im Preset-Menü des gemeinsamen Control-Panels nur noch die Presets anzeigen, die zur Klasse des
aktuell ausgewählten Soundobjekts passen.

### 1.2 Context
Aktuell listet das GUI alle Presets eines Ordners, auch wenn sie für die selektierte
Sound-Klasse ungeeignet sind. Das Laden ist zwar defensiv abgesichert, aber der Workflow wird
unnötig unübersichtlich: Beim Sounddesign und Komponieren sollen im Menü nur sinnvolle Presets
auftauchen.

### 1.3 Related To Intents

* 43 - Live-editierbare Grillen-Sounds (synthetisch + Sample) mit Preset-Bibliothek
* 45 - Preset-Bereich im Control-Panel vollständig bedienbar machen
* 47 - Selektiertes Soundobjekt im Control-Panel solo abhören

## 2.0 What

```gherkin
Feature: Preset-Menü zeigt nur passende Presets

  Scenario: Nur kompatible Presets im Menü
    Given ein ausgewähltes Soundobjekt mit einer konkreten Sound-Klasse
    When das Preset-Menü aufgebaut wird
    Then erscheinen nur Presets, deren soundClass zu diesem Soundobjekt passt

  Scenario: Preset speichern aktualisiert die gefilterte Liste
    Given ein ausgewähltes Soundobjekt
    When ich ein Preset speichere
    Then erscheint es sofort im Menü dieser Sound-Klasse

  Scenario: Auswahlwechsel auf andere Sound-Klasse
    Given ich wechsle auf ein Soundobjekt einer anderen Sound-Klasse
    When das GUI den Objektbereich neu aufbaut
    Then zeigt das Preset-Menü nur die Presets der neuen Sound-Klasse

  Scenario: Laden bleibt defensiv abgesichert
    Given ein manuell ausgewählter oder alter Preset-Name passt nicht zur Sound-Klasse
    When Laden ausgelöst wird
    Then wird kein falsches Preset angewendet
```

## 3.0 How

### 3.1 Implementation Context

* Betroffen sind primär `classes/sg/sounds/SoundPresetLibrary.sc` und
  `classes/sg/gui/SpatialControlPanel.sc`
* `SoundPresetLibrary` bekommt eine gefilterte Listing-Methode anhand von `soundClass`
* Das GUI nutzt diese Methode beim Aufbau/Aktualisieren des Preset-Menüs
* Die bestehende Klassenprüfung beim eigentlichen Laden bleibt zusätzlich erhalten
